Projects

Film Forager

A film app that contains many features! From popular and trending films to a popular star, you can search for movie details afar! The web app was created with React and the data is provided by the Movie Database API.

Old Movies vs New Movies

This or That is a simple "This or That?" game in which you can choose between two thematically related films. I wanted to transform a popular movie list into an interactive experience and this is the result. Check it out, maybe you'll find a cool movie to watch this weekend!

Link Keeper

A simple Chrome extension built with vanilla JavaScript. The extension utilizes local storage to save and set the inputted link or the link in the window's current tab. There is also an option to download the list as a .csv file.

Spartan Fitness

A functional e-commerce shop experience for a gym equipment company. This web app is created with React for the frontend, Node/Express for the server, and MongoDB for the database.
